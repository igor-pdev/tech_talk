Front-end

O front-end de um site é tudo o que o usuário vê ou interage quando visita o site. É responsável pela aparência total de uma experiência online. Embora o termo possa parecer um pouco técnico, os front-ends do site são realmente um encontro diário para quase todos nós.

UX/UI

UX e UI são áreas complementares do design que se referem à experiência e à interface do usuário, respectivamente. A diferença entre as duas é que UX se concentra na experiência do usuário, enquanto UI se preocupa com a interface.

Data Science

A ciência de dados é um campo acadêmico interdisciplinar que usa estatística, computação científica, métodos científicos, processamento, visualização científica, algoritmos e sistemas para extrair ou extrapolar conhecimento e insights de dados potencialmente barulhentos, estruturados ou não estruturados.

Back-end

O desenvolvedor back-end é um profissional especialista no desenvolvimento web. Ele tem a responsabilidade de planejar, programar, testar e manter a estrutura de códigos que faz a interface entre um site, o servidor e o banco de dados. Sua atuação é fundamental para garantir a eficiência e a segurança do sistema.
